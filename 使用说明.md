# 中国象棋 - 使用说明

## 应用概述

这是一款支持蓝牙直连和局域网Wi-Fi对战的中国象棋安卓应用，提供完整的象棋规则实现和多种对战模式。

## 系统要求

- Android 8.0 (API 26) 及以上版本
- 支持蓝牙功能（蓝牙对战模式）
- 支持Wi-Fi功能（局域网对战模式）
- 建议屏幕分辨率：720x1280 或更高

## 安装方法

1. 下载 APK 安装包
2. 在安卓设备上打开 APK 文件
3. 允许安装未知来源应用（如需要）
4. 完成安装

## 功能介绍

### 1. 主界面

启动应用后，主界面提供四个选项：

- **蓝牙对战**：通过蓝牙与其他设备进行对战
- **局域网对战**：通过Wi-Fi局域网与其他设备进行对战
- **单机练习**：本地双人对弈模式
- **设置**：配置游戏参数

### 2. 单机练习模式

#### 操作步骤：
1. 在主界面点击"单机练习"
2. 进入游戏界面，红方先行
3. 点击己方棋子选中（会高亮显示合法移动位置）
4. 点击高亮位置移动棋子
5. 双方轮流操作，直到游戏结束

#### 功能按钮：
- **悔棋**：撤销上一步移动（仅限单机模式）
- **暂停**：暂停游戏计时
- **和棋**：请求和棋
- **认输**：认输结束游戏

### 3. 蓝牙对战模式

#### 操作步骤：

**作为主机（红方）：**
1. 点击"蓝牙对战"
2. 点击"作为主机"
3. 等待从机连接
4. 连接成功后自动进入游戏

**作为从机（黑方）：**
1. 点击"蓝牙对战"
2. 点击"搜索设备"
3. 在列表中选择要连接的主机
4. 连接成功后自动进入游戏

#### 注意事项：
- 确保双方设备蓝牙已开启
- 首次连接可能需要配对
- 主机为红方，先行
- 从机为黑方，后行

### 4. 局域网对战模式

#### 操作步骤：

**创建房间（红方）：**
1. 点击"局域网对战"
2. 点击"创建房间"
3. 等待其他玩家加入
4. 界面会显示本机IP地址

**加入房间（黑方）：**
1. 点击"局域网对战"
2. 点击"搜索房间"自动搜索
3. 或在输入框中输入对方IP地址
4. 点击"连接"
5. 连接成功后自动进入游戏

#### 注意事项：
- 双方设备需连接同一Wi-Fi网络
- 创建房间者显示本机IP，可告知对方直接输入
- 主机为红方，先行
- 客户端为黑方，后行

### 5. 游戏设置

在设置界面可以配置以下选项：

#### 时间设置
- **时间模式**：
  - 无限制：不限制时间
  - 总时间：每方总时间限制（如10分钟）
  - 步时：每步时间限制（如30秒）
- **总时间**：设置每方的总时间（分钟）
- **步时**：设置每步的时间限制（秒）

#### 其他设置
- **音效**：开启/关闭游戏音效
- **震动**：开启/关闭震动反馈

## 游戏规则

### 基本规则
- 红方先行，双方轮流走棋
- 每次只能移动一个棋子
- 不能移动对方的棋子
- 不能移动到已有己方棋子的位置

### 各棋子走法

**将/帅**：
- 只能在九宫内移动
- 每次只能移动一格
- 可以吃掉对方的将/帅（将帅对面）

**士/仕**：
- 只能在九宫内移动
- 只能沿对角线移动一格

**象/相**：
- 不能过河
- 沿对角线移动两格
- 象眼（中间位置）不能被阻挡

**马**：
- 走"日"字形
- 马腿（相邻位置）不能被阻挡

**车**：
- 直线移动，格数不限
- 路径上不能有其他棋子阻挡

**炮**：
- 直线移动，格数不限
- 不吃子时路径上不能有棋子
- 吃子时路径上必须有且只有一个棋子

**兵/卒**：
- 未过河时只能向前移动一格
- 过河后可以向前或左右移动一格
- 不能后退

### 特殊规则

**将军**：
- 一方的将/帅受到对方棋子攻击时，称为"将军"
- 被将军方必须立即解除将军状态

**将死**：
- 一方被将军且无法解除时，称为"将死"
- 被将死方判负

**和棋**：
- 双方同意和棋
- 双方都无法将死对方

## 界面说明

### 游戏界面

**顶部信息栏**：
- 左侧：黑方信息和剩余时间
- 中间：当前回合指示
- 右侧：红方信息和剩余时间

**棋盘区域**：
- 显示完整的中国象棋棋盘
- 棋子以圆形显示，红方红色，黑方黑色
- 选中棋子会高亮显示
- 合法移动位置会显示绿色标记
- 最后一步移动会显示黄色标记

**底部按钮栏**：
- 悔棋、暂停、和棋、认输按钮

## 常见问题

### Q: 蓝牙搜索不到设备？
A: 
1. 确保双方设备蓝牙已开启
2. 确保设备在可发现状态
3. 检查是否授予了位置权限（Android 6.0+需要）
4. 尝试重启蓝牙

### Q: 局域网连接失败？
A:
1. 确保双方连接同一Wi-Fi网络
2. 检查防火墙设置
3. 尝试手动输入IP地址
4. 确保端口8888未被占用

### Q: 游戏卡顿？
A:
1. 关闭其他后台应用
2. 降低屏幕分辨率
3. 重启应用

### Q: 如何悔棋？
A: 单机模式下点击"悔棋"按钮即可。网络对战需要对方同意，暂不支持。
