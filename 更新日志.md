# 中国象棋应用 - 更新日志

## v1.1.0 (2026-04-03)

### 新增功能

#### 1. AI对手（单机练习模式）
- **文件**: [ChessAI.kt](file:///f:/象棋apk/ChineseChess/app/src/main/java/com/chinesechess/game/ChessAI.kt) (273行)
- **算法**: Minimax + Alpha-Beta剪枝
- **难度级别**: 
  - 简单 (搜索深度2)
  - 中等 (搜索深度3)
  - 困难 (搜索深度4)
  - 专家 (搜索深度5)
- **评估函数**: 
  - 棋子基础分值
  - 位置加成
  - 灵活性评估
  - 将军状态评估
- **特性**:
  - 随机因子增加变化性
  - 移动排序优化剪枝效率
  - 500ms延迟模拟人类思考

#### 2. 音效和震动反馈
- **文件**: [SoundManager.kt](file:///f:/象棋apk/ChineseChess/app/src/main/java/com/chinesechess/utils/SoundManager.kt) (273行)
- **音效类型**:
  - 移动音效
  - 吃子音效
  - 将军音效
  - 将死音效
  - 胜利/失败/和棋音效
  - 按钮点击音效
  - 游戏开始音效
  - 悔棋音效
  - 非法移动音效
- **震动模式**:
  - 轻微震动（移动、按钮）
  - 中等震动（吃子）
  - 强烈震动（将军、将死）
  - 节奏震动（胜利、失败）
- **设置**: 支持独立开关音效和震动

#### 3. 对战记录保存与复盘
- **文件**: [GameRecordManager.kt](file:///f:/象棋apk/ChineseChess/app/src/main/java/com/chinesechess/utils/GameRecordManager.kt) (497行)
- **记录保存**:
  - 最多保存50条记录
  - 记录游戏模式、结果、时间、时长
  - 保存完整的移动历史
  - 支持添加备注
- **数据格式**: JSON格式存储于SharedPreferences
- **复盘功能**:
  - 逐步回放
  - 前进/后退
  - 跳转到指定步
  - 调整播放速度
- **统计功能**:
  - 总对局数
  - 胜率统计
  - 各模式对局数
- **导出功能**: 支持导出为PGN格式

#### 4. 网络连接稳定性优化
- **文件**: [NetworkManager.kt](file:///f:/象棋apk/ChineseChess/app/src/main/java/com/chinesechess/network/NetworkManager.kt) (523行)
- **Socket优化**:
  - 禁用Nagle算法（减少延迟）
  - 启用KeepAlive
  - 设置读写超时
  - 优化缓冲区大小
- **心跳机制**:
  - 5秒间隔发送心跳
  - 15秒超时检测
  - 自动重连机制
- **消息队列**:
  - 断线时缓存消息
  - 重连后自动发送
  - 最大50条消息缓存
- **重试机制**:
  - 消息发送失败自动重试
  - 最多3次重连尝试
- **连接状态管理**:
  - CONNECTING - 连接中
  - CONNECTED - 已连接
  - DISCONNECTED - 已断开
  - RECONNECTING - 重连中
  - ERROR - 连接错误

```kotlin

soundManager.playCaptureSound()
```

#### 记录保存
```kotlin

val record = recordManager.createRecordFromGame(...)
recordManager.saveGameRecord(record)

// 复盘

replayController.nextMove()
```
