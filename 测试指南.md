# 中国象棋应用 - 测试指南

## 环境要求

### 必需软件
1. **JDK 17** 或更高版本
2. **Android Studio** (推荐最新稳定版)
3. **Android SDK** (API 26-34)

### 下载地址
- JDK: https://adoptium.net/ 或 https://www.oracle.com/java/technologies/downloads/
- Android Studio: https://developer.android.com/studio

## 项目导入

### 方法一：使用 Android Studio

1. 打开 Android Studio
2. 选择 "Open an existing Android Studio project"
3. 选择 `ChineseChess` 文件夹
4. 等待 Gradle 同步完成（首次可能需要下载依赖，耗时较长）

### 方法二：使用命令行

1. 确保已配置环境变量：
   - `JAVA_HOME` - JDK 安装路径
   - `ANDROID_HOME` - Android SDK 路径

2. 在项目根目录运行：
```bash
# Windows
gradlew.bat build

# Mac/Linux
./gradlew build
```

### 2. 象棋规则测试

#### 2.1 各棋子移动规则
- [ ] **将/帅**：只能在九宫内移动，每次一格
- [ ] **士/仕**：只能沿对角线移动一格
- [ ] **象/相**：不能过河，沿对角线移动两格，象眼不能被挡
- [ ] **马**：走"日"字形，马腿不能被挡
- [ ] **车**：直线移动，格数不限，路径不能被挡
- [ ] **炮**：直线移动，不吃子路径不能有棋子，吃子需隔一个棋子
- [ ] **兵/卒**：未过河只能前进，过河后可前进或左右移动

#### 2.2 特殊规则
- [ ] **将军**：当将/帅受到攻击时提示"将军"
- [ ] **将死**：无法解除将军时游戏结束
- [ ] **将帅对面**：双方将帅在同一直线无遮挡时可以吃掉对方

## 常见问题排查

### 构建失败

#### 问题：Gradle 同步失败
**解决：**
1. 检查网络连接
2. 尝试更换镜像源（阿里云、清华等）
3. 删除 `.gradle` 文件夹后重新同步

#### 问题：缺少 SDK
**解决：**
1. 打开 SDK Manager
2. 安装 Android SDK Platform 26-34
3. 安装 Android SDK Build-Tools

#### 问题：Java 版本不匹配
**解决：**
1. 确保使用 JDK 17 或更高版本
2. 在 `File > Project Structure > SDK Location` 中设置正确的 JDK 路径

### 运行时问题

#### 问题：应用闪退
**排查：**
1. 查看 Logcat 日志
2. 检查是否缺少权限
3. 检查代码空指针异常

#### 问题：蓝牙无法连接
**排查：**
1. 检查蓝牙是否开启
2. 检查是否授予位置权限
3. 检查设备是否配对
4. 尝试重启蓝牙

#### 问题：局域网无法连接
**排查：**
1. 检查双方是否在同一WiFi
2. 检查防火墙设置
3. 尝试手动输入IP
4. 检查端口8888是否被占用
